### Two Pointer（二つのポインタ）とは

Two Pointerは、配列やリストの中で二つのポインタ（インデックス）を使い、探索範囲を狭めながら問題を解く手法です。多くの場合ソート済みの配列に対して使われ、全探索ではO(n²)かかる問題をO(n)まで改善できます。

### 代表的なパターン

1. **両端から狭める**: 左のポインタは先頭、右のポインタは末尾から始め、条件に応じて互いに近づける
   - 例: ソート済み配列で合計がtargetになる2数を探す
2. **同じ方向に進める（スライディングウィンドウ）**: 二つのポインタが両方とも前方にのみ進む
   - 例: 部分配列の合計・長さに関する条件問題

### 例: 二数の和（ソート済み配列）

```python
def two_sum_sorted(arr, target):
    left, right = 0, len(arr) - 1
    while left < right:
        s = arr[left] + arr[right]
        if s == target:
            return left, right
        elif s < target:
            left += 1
        else:
            right -= 1
    return -1, -1
```

### 時間計算量

- 各ポインタは最大でn回しか移動しないため、全体の時間計算量はO(n)
- 全探索（二重ループ）のO(n²)からO(n)へ改善

### 使いどころ

- 配列がソート済み、またはソートしても問題ない場合
- 「二数の和・差」「部分配列」「回文判定」などの問題
- スライディングウィンドウと組み合わせて使われることが多い（条件を満たす最小・最大区間を探す）

### 투포인터란?

투포인터(Two Pointer)는 배열이나 리스트에서 두 개의 포인터(인덱스)를 이용해 탐색 범위를 좁혀가며 문제를 해결하는 기법입니다. 보통 정렬된 배열에서 사용하며, 완전 탐색으로 O(n²)이 걸리는 문제를 O(n)으로 줄일 수 있습니다.

### 대표 패턴

1. **양 끝에서 좁혀오기**: 왼쪽 포인터는 처음, 오른쪽 포인터는 끝에서 시작해 조건에 따라 서로를 향해 이동
   - 예: 정렬된 배열에서 두 수의 합이 target인 쌍 찾기
2. **같은 방향으로 이동 (슬라이딩 윈도우)**: 두 포인터가 모두 앞으로만 이동
   - 예: 부분 배열의 합/길이 조건 문제

### 예시: 두 수의 합 (정렬된 배열)

```python
def two_sum_sorted(arr, target):
    left, right = 0, len(arr) - 1
    while left < right:
        s = arr[left] + arr[right]
        if s == target:
            return left, right
        elif s < target:
            left += 1
        else:
            right -= 1
    return -1, -1
```

### 시간 복잡도

- 각 포인터가 최대 n번 이동하므로 전체 시간 복잡도는 O(n)
- 완전 탐색(이중 반복문) 대비 O(n²) → O(n)으로 개선

### 언제 사용하면 좋은가

- 배열이 정렬되어 있거나 정렬해도 무방할 때
- "두 수의 합/차", "부분 배열", "회문 판별" 등의 문제
- 슬라이딩 윈도우와 함께 자주 등장 (조건을 만족하는 최소/최대 길이 구간 찾기)
